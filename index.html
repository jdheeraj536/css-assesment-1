What is CSS?

  1. CSS stands for Cascading Style Sheet .
  2. CSS describe how html elemnets are to be displayed on screen , paper, or in other media.
  3. Css is not necessary CSS Save a lot of work. it can control the layout of multiple web pages all at once
  4. External stylesheet are stored is CSS file .

CSS can be added to html documents in 3 ways:

  • inline - by using the style attribute inside HTML elements 1
  • internal - by using a style elemnets in the head section
  • External - using a link elemnets to link to an External CSS file .
  • The most common way to add css , is to keep the styles in external CSS files. However , in this tutorial we will inline and internal styles ,because this easier to demonstrate , and easier for you to try it yourself.

When comparing CSS class vs id , the different is that css class applies a style to multiple elements. ID , on other hand , applies a style to one unique elements. ID is also special in that you can use aspecial URl to link directly to an element and it's used by javascript .

in Css, Selector are used to target a specified element or range of elements on a web pages. Once an element has been targeted , a style or set of styles can be applied to the element.

This is a wide range of a selector available. Two of the most Commmonly used are clasds and id . Both are used to target elmemnt to which a styles should be applied.
